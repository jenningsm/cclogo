A potential logo for the Creative Coders group. The logo is generative; each time it is created slightly different.

The large, rounded, colorful arcs communicate the creative aspect of the group while the network of thin, grey lines speaks to the technical side of the group.

This implementation is done in vanilla javascript and writes to a canvas element, but it could be modified to output SVG instead.

Github: github.com/woefij
