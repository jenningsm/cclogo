A logo I created for the Portland Creative Coders group. It is a generative logo; a new version is randomly composed each time it is viewed.

The large, rounded, colorful arcs communicate the creative aspect of the group while the network of thin, metallic arcs speaks to the group's technical side.

This implementation is done in vanilla javascript and draws to a canvas element, but it could be modified to output SVG instead.

Github: github.com/jenningsm/cclogo
